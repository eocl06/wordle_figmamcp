# Quick Start Guide

## How to Run the Application

### Step 1: Install Dependencies
Open terminal in this folder and run:
```bash
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
The terminal will show a URL like:
```
âžœ  Local:   http://localhost:5173/
```

Open that URL in your browser!

## Alternative: If you don't have Node.js

If you don't have Node.js installed, you can:
1. Install Node.js from https://nodejs.org/
2. Then follow steps above

## Troubleshooting

- **Port already in use?** Vite will automatically use the next available port
- **Module not found?** Run `npm install` again
- **TypeScript errors?** Make sure you're using Node.js 18+ and run `npm install`
